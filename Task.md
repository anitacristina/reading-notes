# UNORDERED LIST

Unordered list can be made by preceding one or more lines of text with - or *.

SAMPLE:

"- Item 1"

"* Item 2"

OUTPUT:

* Item 1
* Item 2
  

# ORDERED LIST
 
1. Item 1
2. Item 2
3. Item 3
 * Item 3a
 * Item 3b
